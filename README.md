## Estructura del Repositorio

Este repositorio se divide en dos bloques fundamentales que respaldan la investigación del TFG:

* **`/ConsumeControlTFG`**: Contiene el código fuente completo de la aplicación, la solución de Visual Studio y los recursos de la interfaz (WPF).
* **`/Mediciones`**: Contiene los archivos empíricos (`.CSV`) extraídos durante la fase de validación en diferentes equipos y escenarios de carga (Gaming, Reposo, etc.). Incluye su propio README con los detalles de los sujetos de prueba.
